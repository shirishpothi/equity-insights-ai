# **App Name**: Equity Insights AI

## Core Features:

- Input Fields: Input fields for stock ticker, investment thesis, and goal, mirroring the provided structure.
- AI Stock Analysis: Analyse stocks. Take the data from the ticker requested by the user. Incorporate both fundamental and macroeconomic perspectives based on user inputs (investment thesis and goal), and then tool produces analysis divided into these five sections:
* Fundamental Analysis
* Thesis Validation
* Sector & Macro View
* Catalyst Watch
* Investment Summary
- Report Display: Clear display of the AI-generated equity research report, formatted with markdown and bullet points.
- Collapsible Sections: UI elements to show or hide report sections to simplify viewing, as needed.
- Copy to Clipboard: Implement simple copy-to-clipboard for report content to improve user experience and utility.

## Style Guidelines:

- Primary color: Gradient from black to a deep purple (#301934), conveying a futuristic and sophisticated feel.
- Background color: Dark gray (#222222), providing a modern and immersive backdrop for the content.
- Accent color: Electric violet (#8F00FF), drawing attention to interactive elements and calls to action within the interface.
- Body font: 'Inter', a sans-serif font known for its clean and modern appearance, optimized for readability on screens.
- Headline font: 'Space Grotesk', a sans-serif font, to provide clear hierarchy and a modern touch to headers and titles.
- Use simple, professional icons with a neon or glowing effect for categories/sections to enhance user navigation and comprehension.
- Maintain a clean and structured layout to facilitate easy navigation and readability, mirroring professional financial reports.
- Subtle transitions and loading animations with a futuristic or tech-inspired flair to provide a smooth user experience.
